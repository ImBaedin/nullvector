# Defense Icon Art Style Guide

This folder contains generated defense art for the web UI.

## Visual style

- Isometric sci-fi defense portrait
- Readable at small sizes
- Clean digital painting with crisp edges
- Centered single defense structure on a dark vignette background
- No text, no logos, no watermark

## Consistency rules

- Camera: slight top-down 3/4 isometric view
- Composition: centered structure with clear silhouette
- Palette: steel blue, cyan, slate, restrained amber highlights
- Materials: brushed metal, reinforced panels, subtle emissive strips
- Lighting: cool rim light, soft ambient shadow, controlled glow accents
