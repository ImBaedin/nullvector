# Storage Icon Art Style Guide

This folder contains generated resource storage art for the web UI.

## Visual style

- Isometric sci-fi storage portrait
- Icon-adjacent and readable at small sizes
- Semi-flat digital painting with crisp edges
- Centered single subject on transparent background
- No text, no logos, no watermark

## Consistency rules

- Camera: slight top-down 3/4 isometric view
- Composition: centered object with clear silhouette
- Palette: steel blue, cyan, slate, restrained resource-specific highlights
- Materials: brushed metal, reinforced panels, subtle illuminated strips
- Lighting: cool rim light, soft ambient shadows, subtle glow accents
