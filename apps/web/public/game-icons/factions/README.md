# Faction Portrait Art Style Guide

This folder contains hostile faction portrait art for the web UI.

## Visual style

- Sci-fi character or faction portrait
- Readable at small sizes
- Polished digital painting with strong silhouette
- Centered bust portrait on a dark vignette background
- No text, no logos, no watermark

## Consistency rules

- Composition: shoulders-and-head bust, slight 3/4 angle
- Palette: cool metallic neutrals with controlled faction-colored accents
- Lighting: dramatic rim light with restrained emissive highlights
- Materials: armored plating, cybernetics, reinforced fabrics, subtle UI glows
